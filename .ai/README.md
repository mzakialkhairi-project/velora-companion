# Welcome

Selamat datang di **Velora**.

Folder `.ai` merupakan pusat dokumentasi yang digunakan oleh AI assistant maupun developer untuk memahami project sebelum melakukan perubahan pada source code.

Dokumen di dalam folder ini menjadi **single source of truth** mengenai tujuan project, arsitektur, aturan coding, dan ruang lingkup pengembangan.

---

# About Velora

Velora adalah platform AI mobile yang berfokus pada pengalaman percakapan yang personal.

Project ini bukan sekadar AI Chat, melainkan fondasi untuk membangun:

- AI Companion
- Character Chat
- Roleplay
- Storytelling
- Long-term Memory
- Local AI & Cloud AI Integration

Velora dikembangkan sebagai personal portfolio project dengan fokus pada kualitas arsitektur, pengalaman pengguna, dan maintainability.
