# Project Overview

## Project Name

AI Roleplay Platform

---

## Project Description

AI Roleplay Platform adalah aplikasi mobile berbasis AI yang memungkinkan pengguna berinteraksi dengan berbagai karakter AI secara natural melalui percakapan, roleplay, storytelling, diskusi, brainstorming, dan penggunaan AI sehari-hari.

Project ini merupakan personal project sekaligus portfolio jangka panjang yang berfokus pada kualitas software, pengalaman pengguna, dan eksplorasi teknologi Artificial Intelligence.

Project ini bukan clone ChatGPT.

Tujuan utamanya adalah membangun pengalaman AI yang lebih personal, konsisten, dan menyenangkan.

---

# Vision

Membangun platform AI yang terasa seperti berbicara dengan karakter yang benar-benar hidup.

Bukan sekadar chatbot yang menjawab pertanyaan.

---

# Mission

• Menciptakan pengalaman roleplay berkualitas tinggi.

• Mendukung berbagai model AI lokal maupun cloud.

• Memberikan sistem memory yang konsisten.

• Memiliki Character Card yang fleksibel.

• Menjadi playground untuk mempelajari AI Engineering modern.

• Menjadi portfolio yang menunjukkan kemampuan Software Architecture, Backend, Mobile, dan AI Engineering.

---

# Primary Features

## AI Chat

Percakapan umum menggunakan berbagai model AI.

---

## Roleplay

Pengguna dapat membuat atau memilih karakter AI.

Setiap karakter memiliki:

- personality
- background
- speaking style
- greeting
- memories
- relationship
- world information

---

## Character Cards

Karakter dapat dibuat sendiri.

Karakter dapat diexport.

Karakter dapat diimport.

---

## Model Switching

User dapat mengganti model AI kapan saja.

Contoh:

- Qwen
- Gemma
- Llama
- Claude
- GPT
- Gemini

Pergantian model tidak mengubah UI maupun riwayat chat.

---

## Conversation History

Semua percakapan disimpan.

Percakapan dapat dicari.

Percakapan dapat dihapus.

Percakapan dapat diexport.

---

## Long-term Memory

AI dapat mengingat informasi penting.

Memory dapat:

- ditambahkan
- diedit
- dihapus

---

## World Book

Karakter dapat memiliki dunia sendiri.

Misalnya:

- sekolah
- kerajaan
- perusahaan
- kota
- keluarga

Informasi dunia digunakan saat roleplay.

---

## Prompt Template

Setiap karakter memiliki prompt sendiri.

Prompt dibangun otomatis oleh backend.

User tidak perlu menulis prompt yang panjang.

---

## Persona

User juga memiliki persona.

Contoh:

Nama

Hobi

Profesi

Preferensi

Informasi ini dapat digunakan AI untuk memberikan pengalaman yang lebih personal.
